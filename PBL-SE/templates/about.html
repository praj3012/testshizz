About This Project

The Hand Gesture Recognition project is an AI-based web application designed to detect and interpret various hand gestures using computer vision. It aims to create an intuitive interface that understands user gestures via webcam or image upload, offering real-time recognition and feedback.

This project utilizes OpenCV for video processing, TensorFlow/Keras for gesture classification, and Flask as the backend framework. The frontend is designed to be fully responsive and styled for a modern user experience using HTML, CSS, and JavaScript.

Our goal is to provide a seamless way to interact with machines through natural gestures, enhancing accessibility and user interaction in various domains such as education, gaming, and robotics.

For more technical details, visit the Documentation section.
